# 🤖 مساعد PDF الذكي

تطبيق ويب عربي لتحليل وترجمة ملفات PDF باستخدام الذكاء الاصطناعي (Gemini + DeepSeek)

## ✨ المميزات

- 📄 **تحليل PDF** - استخراج النص من الملفات
- 🌍 **ترجمة فورية** - Gemini API و DeepSeek API
- 🗣️ **شرح باللهجة السودانية** - شرح مبسط بالعامية
- 📚 **بطاقات تعليمية** - Flashcards ذكية
- 🎯 **اختبارات تفاعلية** - أسئلة مع إجابات نموذجية
- 🔊 **نطق النصوص** - قراءة النتائج بصوت
- 🎤 **إدخال صوتي** - اسأل بالصوت
- 🌙 **الوضع الليلي** - مريح للعين
- 💾 **تخزين محلي** - حفظ البيانات في المتصفح

## 🛠️ التقنيات المستخدمة

- HTML5 / CSS3 / TailwindCSS
- JavaScript (ES6+)
- PDF.js - استخراج النص من PDF
- Gemini API - ترجمة وتحليل ذكي
- DeepSeek API - ترجمة وتحليل ذكي (احتياطي)
- Chart.js - رسوم بيانية
- Web Speech API - نطق وإدخال صوتي

## 🚀 طريقة التشغيل

1. حمّل الملف `index.html`
2. افتحه في المتصفح (Chrome / Edge / Firefox)
3. اختر جنسك وادخل اسمك
4. ارفع ملف PDF
5. استمتع بالتحليل الذكي!
